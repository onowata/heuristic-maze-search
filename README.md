# heuristic-maze-search

## 概要

Pythonでヒューリスティック探索を用いた迷路探索プログラムを作成しました。

ゴールまでの推定距離としてマンハッタン距離を利用し、探索候補の優先順位を決定することで、効率的な経路探索を行うことを目的として作成しました。

また、探索結果をコストマップに反映させる仕組みを実装し、探索過程の情報も経路候補の評価に活用できるよう工夫しました。

## 使用言語

* Python
* Jupyter
* NotebookNumPy

## 工夫した点

* ゴールまでの推定距離としてマンハッタン距離を利用し、効率的な探索を行えるようにした。
* 授業内で実装したユークリッド距離による探索結果と比較することで、それぞれの評価方法の特徴を確認できるようにした。（詳細な比較結果はNotebook内に記載）
* 探索成功時の経路情報を学習するコストマップを実装し、経路候補の評価値に活用できる仕組みを導入した。

## 今後の改善点

* A*探索など他の探索アルゴリズムとの性能比較
* 学習時のゴール地点をランダムに設定し、学習処理を自動化する
* より大規模な迷路への対応
* 迷路の自動生成機能の実装
